# Advent of Code 2025

Run solutions using [pixi](https://pixi.sh):

```bash
pixi run run day{{your_day}} {{Release|Debug}} {{extra_args}}
```

For example, to run day 1:

```bash
pixi run run day1
```

Pixi will automatically download all dependencies.
To see the available tasks:

```bash
pixi task list
```
